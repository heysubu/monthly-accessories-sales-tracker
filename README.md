# üìä Monthly Accessories Sales Performance Tracker

An automated Excel-based sales performance tracking system that analyzes monthly accessory sales data, calculates salesperson achievements, and generates incentive reports. Designed for automotive dealerships to track accessory sales (450 X, Side Steps, TPMS, Software Updates) and calculate proper incentive amounts based on individual performance.

## üìã Overview

This system solves a critical business challenge: **How to accurately track individual salesperson performance and calculate fair incentives based on monthly accessory sales achievements.**

The system helps management and sales teams by:
- **Automated Performance Tracking**: Monitor each salesperson's monthly sales across multiple accessory types
- **Instant Data Processing**: Import DMS data and get results in seconds
- **Visual Dashboards**: Interactive charts showing performance trends and achievements
- **Fair Incentive Calculation**: Automatic calculation of incentive amounts based on monthly targets
- **Month-wise Analysis**: Track performance trends over time for each salesperson

### How to Use

**Step 1: Export Data from DMS**
- Open DMS Operator Software (Excellence Software)
- Generate monthly sales report
- Export as CSV file

**Step 2: Import to Tracker**
- Open the tracker file
- Navigate to "Monthly Sales" sheet
- Copy data from CSV
- Paste into the designated area (starting from row 2)

**Step 3: View Results**
- Dashboard automatically updates
- Check individual salesperson performance
- Review incentive calculations
- Generate reports for management presentation

**Step 4: Analyze Performance**
- Use filters to view specific months or salespeople
- Compare performance across team members
- Identify top performers and improvement areas
- Track achievement against monthly targets
